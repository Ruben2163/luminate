What is LumOS?

LumOS is a lightweight operating system written in Rust, designed for speed, efficiency, and a fully customizable experience. Unlike Linux-based systems, LumOS offers a completely unique and custom architecture.

When Will Development Start?

Development begins in January 2025. You can contribute to LumOS before and during this time to help shape its future.

Key Features

  • Custom Rust Bootloader
  • Lightweight Rust Kernel
  • Command-Line Interface

A GUI might be added in the future, but initially, it will be command-line-based.

Warnings

For Virtual Machine Use Only: LumOS is intended for use in VirtualBox or VMware.

Security Limitations: We are not security experts, so use LumOS at your own risk.

Contributing

LumOS is a massive project. We welcome contributions of all sizes, from small fixes to major features. Every bit of help is appreciated!

Why Help?

  • Accelerate new releases.
  • Shape features with your ideas.
  • Be part of a groundbreaking OS project.

Why Rust?

Rust is a fast, low-level language that simplifies development compared to assembly. It enables:

  • Faster, more secure programming.
  • Greater collaboration within the community.

Our Goals

We aim to:

  • Learn Rust and system-level programming.
  • Make a basic and customizable OS for tinkerers.
  • Create something new and innovative.

Who Should Use LumOS?

We recommend LumOS for tinkerers and contributors who enjoy exploring unique, early-stage projects.
